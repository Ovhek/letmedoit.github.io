Una clase ViewModel que conecta la vista con el repositorio de Services. Proporciona objetos LiveData para mostrar el estado actual de las operaciones. Todas las funciones de esta clase actuan sobre la base de datos. Utiliza los UseCase para comunicarse con el repositorio
