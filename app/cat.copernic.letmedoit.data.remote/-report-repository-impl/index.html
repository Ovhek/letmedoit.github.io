class ReportRepositoryImpl @Inject constructor(val reportsCollection: CollectionReference) : ReportRepository

Clase que implementa la interfaz ReportRepository que permite conectarse a la base de datos remota de Firebase y realizar operaciones de CRUD con los reportes. Utiliza la librería de coroutines de Kotlin para manejar operaciones asíncronas y emitir flujos de datos (flow) para informar el estado de las operaciones.
