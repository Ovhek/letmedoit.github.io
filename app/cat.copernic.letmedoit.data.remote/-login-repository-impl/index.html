LoginRepositoryImpl

class LoginRepositoryImpl @Inject constructor(auth: FirebaseAuth, usersCollection: CollectionReference) : LoginRepository

Clase que implementa la interfaz LoginRepository que permite conectarse a la base de datos remota de Firebase y realizar operaciones de login y registro. Utiliza la librería de coroutines de Kotlin para manejar operaciones asíncronas y emitir flujos de datos (flow) para informar el estado de las operaciones.
