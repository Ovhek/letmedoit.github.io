sealed class DataState<out R>

Sealed Class que representa estados de una petición --> éxito, error, carga y finalización. Interfaz que nos permite representar jerarquias restringidas donde un objeto solo puede ser de un tipo de las dadas. Muy util para la programación funcional --> permite cocnatenar funciones en una sola instrucción. Básicamente es como un enum. Permite tener una clase con un número especifico de clases.
